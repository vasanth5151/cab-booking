Google Maps API Required

To use the location autocomplete and route visualization features, you need to:

  1. Get a Google Maps API key from Google Cloud Console
  2. Enable the following APIs:
    • Maps JavaScript API
    • Places API
    • Directions API
    • Distance Matrix API
  3. Replace 'YOUR_API_KEY' in the script tag with your actual API key
